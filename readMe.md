# Introduce

-   Website is cloned from https://phongtro123.com/
-   Frontend: NextJs
-   Backend: NestJs
-   Database: mysql (Sequelize)
-   Image: cloudinary (https://cloudinary.com/)

# Features

-   Filter
-   Pagination
-   CRUD Posts
-   Authentication with JWT
-   Persist login - protected routes
